### 🚀 About Me

Cybersecurity analyst with hands-on experience across Threat Hunting, Cyber Threat Intelligence (CTI), Digital Forensics &amp; Incident Response (DFIR), Security Assessment, and Governance, Risk &amp; Compliance (GRC). I work across enterprise environments — Active Directory, secure architecture, network segmentation, and Zero Trust — using tools like Wazuh, Nessus, MITRE ATT&amp;CK, VirusTotal, and Docker on Kali, Windows, and Linux. Currently completing cybersecurity training at Moringa School and preparing for CompTIA certification. Passionate about understanding attacker behavior and building systems that are secure by design.

### 🎯 Core Competencies

`Threat Hunting` · `Cyber Threat Intelligence` · `Digital Forensics (DFIR)` · `Incident Response` · `Threat Modeling` · `Secure SDLC` · `Security Assessment` · `Vulnerability Management` · `System Hardening` · `Secure Architecture` · `GRC` · `SOC Operations` · `Active Directory` · `Zero Trust & Microsegmentation`
